java语言的特点：

​ 1.跨平台（一次开发，到处运行）

java开发环境：

​ jdk：java开发工具包（开发人员必须安装）

​ jre：java运行环境（运行java程序必须安装）

​ 注意：java程序最终是运行在jvm虚拟机上的，不同的操作系统上可以安装其对应版本的jvm，这样就实现了跨平台
